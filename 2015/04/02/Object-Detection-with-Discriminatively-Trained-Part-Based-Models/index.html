Object Detection with Discriminatively Trained Part Based Models

文章目录
  1. 1. Highlight
  2. 2. Models

P. Felzenszwalb, R. Girshick, D. McAllester, D. Ramanan. Object Detection with Discriminatively Trained Part Based Models. TPAMI. 2010. pdf

最近看代码搞不清楚Latent Structured SVM是怎么弄的. 追根溯源又回到DPM. 做个小结后再去看S-SVM.

Highlight

  1. 提出Mixtures of Deformable Part Based Models;
  2. 提出了Latent SVM (LSVM) 来训练含hidden variable的模型 (因为本文的part location并没有标注, 在训练时只能当作latent/hidden variable);
  3. Deformation Cost: 每个part相对root都有一个理想位置, 当产生偏移时, 需要给予一定的惩罚. 这个思想在后续工作中多有用到.
  4. Training 阶段用通过mining hard negative examples 来减少潜在的false positives.
  5. 通过对HOG进行主成分分析对其进行降维, 实现HOG快速计算.
