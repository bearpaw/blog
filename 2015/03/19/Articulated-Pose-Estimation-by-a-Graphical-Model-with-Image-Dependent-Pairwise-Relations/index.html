Articulated Pose Estimation by a Graphical Model with Image Dependent Pairwise Relations

Chen, Xianjie, and Alan L. Yuille. “Articulated pose estimation by a graphical model with image dependent pairwise relations.” Advances in Neural Information Processing Systems. 2014.

本文在YANG Yi工作的基础上, 利用CNN学习part detector以及pairwise part relations, 并利用图模型进行Inference, 从而估计出最终的pose.

Motivation

本文方法的出发点是认为, 一个合适大小的local patch可以告诉我们两个信息. 1) 我们可以通过合适的local patch判断它是哪一个body part; 2) 在知道local patch是哪一个part后, 我们还可以看出其相邻的part可能出现的位置. 如图1所示. 因此, 基于part model的方法一般要具备part detection以及描述pairwise relation的能力.

Model

和传统方法类似, 作者采用了一个图模型 (tree structure) 来表示human pose. 传统模型是有向图, 而本文更像是一个无向图. 图模型的full score定义如下

$$F(l, t|I) = \sum_{i \in V} U(l_i | I) + \sum_{(i,j)\in E} R(l_i, l_j, t_{ij}, t_{ji} | I) + w_0.$$
其中$w_0$是偏置.

Unary term


所谓的Unary term其实就是一个part detector在图像上扫描得到的score map. 每一个score表示local patch $I(l_i)$被分类为第i类的分数. 数学表达式如下:
$$U(l_i | I) = w_i \phi(i | I(l_i); \theta).$$

其中$\phi(i | I(l_i); \theta)$是apprearance term, $\theta$表示CNN学出来的参数. $w_i$是structure-svm学出来的参数.

Image Dependent Pairwise Relational (IDPR) Terms

假如给定一个patch以part $i$ 为中心, 那么根据这个patch, IDPR能给出 $i$ 的所有neighbor $j$ 的相对位置 (这个相对位置在程序里是用 $(x_i-x_j, y_i-y_j)$ 来描述). 作者首先将 $(i, j)\in E$ 的相对位置用K-means聚类, 即

$$t_{ij} \in \{1, \cdots, T_{ij}\}.$$
我们能获得聚类中心$\mathbf{r}^{t_{ij}}_{ij}$, 再加上quadratic deformation $\psi(dx, dy) = [dx, dx^2, dy, dy^2]^T$, 就得到了两个part i, j之间的pairwise relational score:
$$\begin{eqnarray} R(l_i, l_j, t_{ij}, t_{ji} | I) &=& \langle\mathbf{w}^{t_{ij}}_{ij}, \mathbf{\psi}(l_j-l_i-r^{t_{ij}}_{ij})\rangle + w_{ij}\varphi(t_{ij}|I(l_i); \theta) \\ &+& \langle\mathbf{w}^{t_{ji}}_{ji}, \mathbf{\psi}(l_i-l_j-r^{t_{ji}}_{ji})\rangle + w_{ji}\varphi(t_{ji}|I(l_j); \theta) \end{eqnarray}$$
其中$\varphi(t_{ij}|I(l_i); \theta)$ 就是IDPR term.

如何计算Unary中的appearance term和Pairwise relation中的IDPR term


本文的任务是, 给定一个图像块, 既要能判断它是 part $i$ 的概率, 也要在此基础上确定它与它所有的neighboring part的relationship $t_{ij}$. 两者都可以用同一个CNN output来计算得到.

作者用概率来formulate这个问题. 首先引入随机变量$c$, $c = 0$ 表示当前patch是negtive, $c = i \in \{1, \cdots, K\}$ 表示当前patch是第$i$个part. 同时, 还定义了随机变量 $\mathbf{m}_{c\mathbb{N}(c)}$来表示$c$所属于哪个mixture type, 注意$\mathbf{m}_{c\mathbb{N}(c)}$ 的 state space 是 $\mathbb{M}_{c\mathbb{N}(c)}$. 假如$c=i$ 只有一个neighbor, 那么状态空间为 $\mathbb{M}_{i\mathbb{N}(i)} = \{1, \cdots, T_{ij}\}$; 若有两个neighbor $j, k$, 那么状态空间为$\mathbb{M}_{i\mathbb{N}(i)} = \{1, \cdots, T_{ij}\}\times \{1, \cdots, T_{ik}\}$, 依此类推. 对Negtive而言, $\mathbb{M}_{0\mathbb{N}(0)} = 0$.

这样, 总的状态空间就是 $\mathbb{S} = \cup_{c=0}^K \{c\}\times \mathbb{M}_{c\mathbb{N}(c)}$, 总的状态数是 $|\mathbb{S}| = \sum_{c=0}^{K} | \mathbb{M}_{c\mathbb{N}(c)}|$. 在CNN中, 就变成了对local image patch 进行 $|\mathbb{S}|$ 类分类的问题, 最后每一类的概率 $p(c, \mathbf{m}_{c\mathbb{N}(c)} | I(l_i); \theta)$ 由softmax计算得到, 这里的$\theta$就是CNN的模型参数.

最后, appearance 和 IDPR 可以很简单的通过marginalization得到:

$$\begin{eqnarray} \phi(i | I(l_i)) &=& \log(p(c=i | I(l_i))) \\ \varphi(t_{ij} | I(l_i)) &=& \log(p(m_{ij} = t_{ij} | c=i, I(l_i))). \end{eqnarray}$$

(这里还要仔细看下代码是怎么做marginalization的.)

Inference

Inference就是要找到最优的part的位置和所属于的mixture, $(\mathbf{l^*, t^*}) = \arg\max_{\mathbf{l, t}} F(\mathbf{l, t|I})$. 由于本文pose的模型是tree structure, 可以用Dynamic programming 来求解. 有关DP的内容, 参见:

  1. 动态规划：从新手到专家
  2. 一个教程涉及了dp及backtrack : http://www.hawstein.com/posts/dp-knapsack.html

求解过程如下: 定义$\mathbb{K}(i)$表示第 i 个part的所有children, 定义 $S_i(l_i | I)$ 表示以 i 为root的子树所能得到的maximum score, 这个score 也就是DP里面的状态, 且状态转移方程定义如下:

$$S_i(l_i | I) = U(l_i | I) + \sum_{k \in \mathbb{K}(i)} \max_{l_k, t_{ik}, t_{ki}} \{R(l_i, l_k, t_{ik}, t_{ki} | I) + S_k(l_k |I)$$

DP算法最后算在tree的root时, 能够得到最大score. 通过backtrack算法就能得到各个part对应的location和mixture type $(l_i, t_{ij})$.

快速求解 pairwise term

这里用了generalized distance transform(Pictorial Structures for Object Recognition), 暂时还没有去了解是什么, 不过这篇文章很经典啊.

Learning

根据前面的介绍, 主要有三组参数要学:

  1. mean relative positions: $r_{ij}^{t_{ij}}$ 是对 part i, j 的相对位置进行k-means聚类得到的聚类中心.
  2. Parameters of Image Dependent Terms: 也就是$\theta$ 其实就是CNN的参数.
  3. Full score weight: 在positive image上的pose本来就有人工标注的location信息, 然后mixture type也通过聚类得到了: $(I^n, l^n, t^n)$. 这样就能把 image dependent terms 和 deformation features拼成长向量, 训练structure SVM.
